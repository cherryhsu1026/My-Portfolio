# 線上商城 （MarketPlace)

涵蓋商家商品上架、管理、訂單處理及消費者下訂流程，並整合線上金流付款，確保交易流程自動化與安全性。
原始系統因公司解散已下線，這裡以文件與技術說明方式呈現。

---
- **技術選型**：ASP、VBScript、MS SQL、Javascript、CSS、DHTML
- **系統設計**：從商品上架、訂單管理到金流串接
- **開發內容**：獨立規劃並開發整體商城流程
- **成效**：實現商品交易流程自動化，整合線上付款，提升交易安全性
